## 確認依頼事項

### 1. 金融商品取引法関連

**懸念点:**
- 「損得評価機能」「シミュレーション」が投資助言に該当しないか
- 「節税効果」の表現が優良誤認に該当しないか

**現在の対応:**
- 免責事項を各所に配置
- 「概算」「目安」「試算」という表現を使用
- 専門家への相談を推奨

**確認いただきたい点:**
- [ ] 現在の免責事項の文言・配置は十分か
- [ ] 追加すべき表現や警告文はあるか
- [ ] 「節税」という表現は使用可能か

---

### 2. 税理士法関連

**懸念点:**
- 「法令検索機能」「理解度チェック」が税務相談に該当しないか

**現在の対応:**
- 「情報提供目的」であることを明示
- 「税務相談ではない」旨を記載
- 税理士への相談を推奨

**確認いただきたい点:**
- [ ] 現在のサービス内容は税理士法に抵触しないか
- [ ] 広報表現に問題はないか

---

### 3. 景品表示法関連

**懸念点:**
- 「日本初」等の表現が不当表示に該当しないか
- シミュレーション結果の表示が優良誤認に該当しないか

**現在の対応:**
- 「日本初」表現は代替案を準備
- 数値の保証表現は回避

**確認いただきたい点:**
- [ ] 「e-Gov APIを活用した先進的なサービス」は使用可能か
- [ ] シミュレーション結果の表示方法に問題はないか

---

### 4. e-Gov API利用に関する表現

**懸念点:**
- 「政府公式API」という表現が政府との関係を誤認させないか
- 政府のブランドを不当に利用していないか

**現在の対応:**
- 「政府が公開している」と表現
- 当社は政府機関ではないことを明示

**確認いただきたい点:**
- [ ] e-Gov APIの利用規約に違反していないか
- [ ] 広報表現に問題はないか

---

### 5. SNS投稿の文字数制限対応

**懸念点:**
- X（280文字制限）では完全な免責事項を記載できない

**確認いただきたい点:**
- [ ] 短縮版免責事項の最低限必要な文言
- [ ] 免責事項を別ページへのリンクで代替可能か

---

### 6. インフルエンサーマーケティング

**確認いただきたい点:**
- [ ] #PR表記の必須要件
- [ ] インフルエンサーへの依頼時の必須注意事項
- [ ] 投稿内容の事前チェック義務の有無

## 想定リスクと対応方針案

### リスク1: ユーザーからの損害賠償請求

**シナリオ:** シミュレーション結果を信じて投資判断し、損失が発生

**対応方針案:**
- 利用規約での免責条項
- サービス内での繰り返し警告
- 専門家相談の強力な推奨

**CLO確認:** この対応で法的に十分か？

---

### リスク2: 税理士団体からの抗議

**シナリオ:** 「税理士業務を侵害している」との抗議

**対応方針案:**
- 情報提供サービスであることの明確化
- 税理士との連携機能の検討
- 事前の業界団体への説明

**CLO確認:** 事前の関係構築は必要か？

---

### リスク3: 行政指導

**シナリオ:** 金融庁または消費者庁からの指導

**対応方針案:**
- 免責事項の充実
- 広告表現の慎重な選択
- 相談窓口の設置

**CLO確認:** 追加で必要な対応は？
