# LifePlan Navigator ユーザー受け入れテスト（UAT）計画書

## 1. テスト概要

### 1.1 目的
Phase 2（法令API統合）およびPhase 3（機能拡張）の全機能が、ビジネス要件とユーザー期待を満たしているかを検証する。

### 1.2 スコープ
| フェーズ | 機能 | テスト対象 |
|---------|------|-----------|
| Phase 2 | 法令API統合 | 法令検索、パーソナライズ表示 |
| Phase 3 | オンボーディング | 全10ステップ（財務情報含む） |
| Phase 3 | 設定画面 | 全項目編集機能 |
| Phase 3 | 理解度チェック | クイズ機能、損得評価 |
| Phase 3 | ナビゲーション | タブ切り替え、CTAカード |

---

## 2. テストシナリオ

### 2.1 オンボーディングフロー

#### TC-ONB-001: 新規ユーザー登録完了
**前提条件**: 未登録ユーザー
**手順**:
1. `/register`にアクセス
2. メールアドレス、パスワード、名前を入力
3. 「登録」ボタンをクリック
4. オンボーディング画面に遷移

**期待結果**:
- 登録成功メッセージ表示
- オンボーディング画面（ステップ1）に遷移

#### TC-ONB-002: オンボーディング全ステップ完了
**前提条件**: 登録済みユーザー
**手順**:
1. 基本情報入力（名前、生年月日、性別）
2. 住所入力（都道府県、市区町村）
3. 職業・年収入力
4. 婚姻状況・子ども情報入力
5. 住居形態入力
6. 資産・家計情報入力（★新規）
7. 将来の予定選択
8. 目標選択
9. 好きな動物選択
10. 完了確認

**期待結果**:
- 各ステップが順次表示
- 入力データが保存される
- ダッシュボードに遷移

---

### 2.2 設定画面

#### TC-SET-001: 全セクション表示確認
**前提条件**: オンボーディング完了ユーザー
**手順**:
1. ダッシュボードから「設定」タブをクリック
2. 「プロフィール設定」パネルを開く
3. 8つのセクションタブを確認

**期待結果**:
- 基本情報、住所、仕事、家族構成、資産・家計、将来の予定、目標、好きな動物タブが表示

#### TC-SET-002: 財務情報編集
**前提条件**: オンボーディング完了ユーザー
**手順**:
1. 設定画面で「資産・家計」タブをクリック
2. 貯蓄額を変更
3. 「保存」ボタンをクリック

**期待結果**:
- 変更が保存される
- 保存完了メッセージ表示

---

### 2.3 理解度チェック（クイズ）

#### TC-QUIZ-001: クイズ開始から結果表示
**前提条件**: オンボーディング完了ユーザー
**手順**:
1. ダッシュボードから「理解度チェック」タブをクリック
2. 質問に回答（5問）
3. 結果画面を確認

**期待結果**:
- パーソナライズされた質問が表示
- 正答/誤答のフィードバック表示
- 結果画面に正答率、損得評価、アドバイスが表示

#### TC-QUIZ-002: 損得評価表示
**前提条件**: クイズ完了
**手順**:
1. クイズ結果画面を確認
2. 「活用可能な年間節約額」を確認
3. 「潜在的な損失額」を確認（誤答がある場合）

**期待結果**:
- 金額が正しく計算・表示される
- アドバイスが誤答内容に基づいている

---

### 2.4 法令検索（Phase 2）

#### TC-LAW-001: キーワード検索
**前提条件**: ログイン済みユーザー
**手順**:
1. ダッシュボードの「関連する法令・制度」セクションを確認
2. 検索ボックスにキーワードを入力
3. 検索結果を確認

**期待結果**:
- 関連する法令が表示される
- 法令タイトルとサマリーが表示

#### TC-LAW-002: パーソナライズ表示
**前提条件**: プロフィール設定済みユーザー
**手順**:
1. ダッシュボードにアクセス
2. 「関連する法令・制度」セクションを確認

**期待結果**:
- ユーザーの属性（年齢、職業、家族構成）に基づいた法令が表示
