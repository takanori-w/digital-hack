# LifePlan Navigator - UX設計書

## 1. プロダクト概要

### 1.1 コンセプト
「年1-2回の利用でも価値を感じるお得情報プラットフォーム」

### 1.2 コアバリュー
- **即時性**: 今すぐ使えるお得情報をワンタップで
- **パーソナライズ**: ライフステージに合わせた最適提案
- **比較価値**: 同年収帯との比較で「気づき」を提供
- **プロアクティブ**: 法改正・制度変更を先回り通知
