# LifePlan Navigator - ローンチプラン

## 1. エグゼクティブサマリー

### 1.1 ローンチ概要

| 項目 | 内容 |
|------|------|
| サービス名 | LifePlan Navigator（ライフプランナビゲーター） |
| ローンチ形態 | ソフトローンチ → 正式ローンチ |
| ソフトローンチ日 | 2025年1月中旬（年末調整・確定申告シーズン） |
| 正式ローンチ日 | 2025年2月上旬 |
| 初期ターゲット | 首都圏在住・28-38歳・年収400-800万円 |
| ローンチ目標 | ソフトローンチ: 1,000人 / 正式ローンチ後1ヶ月: 5,000人 |
