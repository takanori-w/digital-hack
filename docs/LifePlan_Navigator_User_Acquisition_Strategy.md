# LifePlan Navigator - ユーザー獲得戦略

## 1. エグゼクティブサマリー

### 1.1 戦略概要
LifePlan Navigatorは「年1〜2回の利用で価値を実感できるお得情報プラットフォーム」として、低頻度利用を前提としたユーザー獲得・維持戦略を展開します。

### 1.2 目標指標（KGI/KPI）

| 指標 | Phase 1（6ヶ月） | Phase 2（12ヶ月） | Phase 3（24ヶ月） |
|------|------------------|-------------------|-------------------|
| 登録ユーザー数 | 10,000人 | 50,000人 | 200,000人 |
| 月間アクティブ率 | 15% | 18% | 20% |
| 年間リテンション | 50% | 60% | 70% |
| オーガニック獲得比率 | 30% | 50% | 65% |
| CAC（顧客獲得コスト） | ¥800 | ¥500 | ¥300 |
| NPS | +30 | +40 | +50 |
