## 政府公式APIと連携したライフプラン支援サービス「LifePlan Navigator」正式リリース
〜 e-Gov法令APIを活用し、信頼性の高い法令情報をパーソナライズして提供 〜

---

### リリース概要

株式会社〇〇（本社：東京都〇〇区、代表取締役：〇〇）は、ライフプラン支援サービス「LifePlan Navigator」のバージョン1.0.0を正式リリースいたしました。

本サービスは、政府が提供するe-Gov法令APIと連携し、補助金・税制優遇・各種制度に関する情報を、ユーザーのライフステージに応じてパーソナライズして提供する日本初のサービスです。

---

### 新機能ハイライト

#### 1. 法令検索機能（e-Gov API連携）

政府公式のe-Gov法令APIと連携し、最新の法令情報を提供します。

- 所得税法、相続税法、国民健康保険法等の主要法令に対応
- ユーザーの関心事項に関連する条文を自動抽出
- 法改正時の自動通知機能

**※ 本機能で提供する情報は一般的な情報提供を目的としており、法的助言ではありません。**

#### 2. 理解度チェック機能

制度の理解を深めるためのインタラクティブな学習機能です。

- 各制度の重要ポイントを問答形式で確認
- 理解度に応じた追加情報の提供
- 専門家への相談が推奨されるケースの案内

#### 3. 損得評価機能

各種制度の活用による概算効果を試算します。

- iDeCo、NISA、ふるさと納税等の節税効果シミュレーション
- 住宅ローン控除の概算計算
- 医療費控除・セルフメディケーション税制の比較

**※ シミュレーション結果は一定の前提条件に基づく試算であり、実際の効果を保証するものではありません。最終的な判断は税理士等の専門家にご相談ください。**

#### 4. パーソナライズダッシュボード

ユーザー属性に基づき、最適な情報を一画面に集約します。

- 潜在的な節約可能額の概算表示
- 優先度の高いネクストアクションの提案
- 同年収帯との統計比較（匿名化データに基づく）

### 重要な免責事項

本サービスは情報提供を目的としており、以下の行為を行うものではありません。

- 金融商品取引法に基づく投資助言
- 保険業法に基づく保険募集
- 税理士法に基づく税務相談
- 弁護士法に基づく法律相談

具体的な判断は、必ず税理士、ファイナンシャルプランナー、弁護士等の専門家にご相談ください。

## 【CMO内部メモ】CLO法務確認依頼事項

以下の表現について、法務確認をお願いします：

1. **「日本初のサービス」**
   - 競合調査に基づく表現ですが、景品表示法上の問題がないか確認
   - 代替案：「e-Gov APIを活用した先進的なサービス」

2. **「節税効果シミュレーション」**
   - 税理士法との抵触がないか確認
   - 免責事項の記載で十分か

3. **「潜在的な節約可能額」**
   - 優良誤認にならないか確認
   - 「概算」「目安」等の表現追加の要否

4. **免責事項の配置**
   - 現在の配置・文言で十分か
   - より目立つ位置への配置が必要か

5. **e-Gov API利用の記載**
   - 政府サービスとの関係性の記載方法
   - 誤解を招く表現がないか
