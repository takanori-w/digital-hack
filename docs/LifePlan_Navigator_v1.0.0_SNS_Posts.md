## 想定されるユーザーコメントへの対応

### ポジティブ反応

**コメント例:** 「これ便利そう！」「使ってみます！」

**対応:**
```
ありがとうございます！
ぜひお試しください。
ご不明点があればお気軽にDMでお問い合わせください。
```

### 機能に関する質問

**コメント例:** 「〇〇機能はありますか？」

**対応:**
```
お問い合わせありがとうございます。
現在は〇〇には対応しておりませんが、
ご要望として開発チームに共有いたします。
今後のアップデートにご期待ください！
```

### 批判的コメント

**コメント例:** 「信頼できるの？」「本当に正確？」

**対応:**
```
ご質問ありがとうございます。
本サービスは政府公式のe-Gov APIと連携し、
正確な法令情報の提供に努めております。

なお、本サービスは情報提供を目的としており、
具体的な判断は税理士等の専門家への
ご相談をお勧めしております。
```

### 法的なクレーム

**対応:** DMに誘導し、カスタマーサポート→CLO確認へエスカレーション

```
ご連絡ありがとうございます。
詳細をお伺いしたいので、
DMにてご連絡いただけますでしょうか。
担当者より改めてご連絡いたします。
```

## 【CMO内部メモ】CLO確認依頼

1. **免責事項の文字数制限対応**
   - X（280文字制限）での免責事項の省略可否
   - 最低限必要な文言の確認

2. **「節税」という表現**
   - 広告での使用可否
   - 代替表現の検討（「税負担の軽減」等）

3. **シミュレーション結果の表現**
   - 「概算」「目安」「試算」の使い分け
   - 数値の例示（「最大〇万円」等）の可否

4. **インフルエンサーマーケティング**
   - 依頼時の必須注意事項
   - 投稿内容の事前チェックフロー
